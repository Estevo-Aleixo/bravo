update_time(self)

source code 

Update the in-game timer.

The timer goes from 0 to 24000, both of which are high noon. The clock increments by 20 every second; this method increments by 200 and should be called every 10 seconds. Days are 20 minutes long.

The day clock is incremented every in-game day, which is every 20 minutes. The day clock goes from 0 to 360, which works out to a reset once every 5 days. This is a Babylonian in-game year.
