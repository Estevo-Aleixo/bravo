ibeta – Interfaces¶

The ibeta module holds the interfaces required to implement plugins and hooks.

Plugins¶

class beta.ibeta.IAuthenticator(name, bases=(), attrs=None, __doc__=None, __module__=None)¶

Interface for authenticators.

Authenticators participate in a two-step login process with a handshake.

class beta.ibeta.ICommand(name, bases=(), attrs=None, __doc__=None, __module__=None)¶

Interface for commands.

Commands may be called from the console or from chat, so they must be fairly stateless.

class beta.ibeta.ISeason(name, bases=(), attrs=None, __doc__=None, __module__=None)¶

Seasons are transformational stages run during certain days to emulate an environment.

class beta.ibeta.ITerrainGenerator(name, bases=(), attrs=None, __doc__=None, __module__=None)¶

Interface for terrain generators.

Hooks¶

class beta.ibeta.IDigHook(name, bases=(), attrs=None, __doc__=None, __module__=None)¶

Hook for actions to be taken after a block is dug up.
